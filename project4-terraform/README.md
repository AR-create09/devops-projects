# Project 4 - Terraform AWS Infrastructure

## Tools Used
- Terraform
- AWS S3
- AWS CLI

## What I Built
Provisioned real AWS cloud infrastructure using
Terraform (Infrastructure as Code):
- S3 bucket with versioning enabled

## Terraform Commands Used
terraform init
terraform plan
terraform apply
terraform destroy

## How to Run
1. Configure AWS: aws configure
2. Init: terraform init
3. Plan: terraform plan
4. Apply: terraform apply
